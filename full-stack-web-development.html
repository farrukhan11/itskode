Course Overview

This 1-year full stack web development bootcamp is designed to give students a comprehensive introduction to the world of web development. Students will learn the fundamental concepts and technologies used in front-end and back-end web development, and will gain hands-on experience building full stack web applications.

The course is divided into three main sections:

Section 1: Introduction to Front-End Development (4 months)

In the Introduction to Front-End Development module of this course, students will embark on a comprehensive journey into the world of front-end web development. This module serves as the foundation for building a strong skill set in front-end development practices and techniques. Throughout this module, participants will explore various topics that form the building blocks of front-end development. The topics covered include:

Topics covered in this section include:

  • Basics of Web
  • Front End Basics
  • Mastery in HTML5
  • Mastery in CSS3
  • Responsive Web Design
  • Bootstrap (Latest Version)

Section 2: Advanced Front-End Concepts (4 months)

In the Advanced Front-End Concepts module of this course, participants will explore a comprehensive range of topics aimed at enhancing their expertise in front-end development. Throughout this module, students will delve into advanced concepts and tools used in modern front-end programming.

Topics covered in this section include:

  • Introduction to JavaScript
  • JavaScript Fundamentals
  • Advanced JavaScript Concepts
  • Document Object Model (DOM)
  • ES6 Features and Syntax
  • ReactJS
  • React Native for Mobile App Development

By the end of this section, students will have a strong understanding of front-end web development and will be proficient in building full-stack web applications using a variety of front-end tools and technologies. Students will also have the skills necessary to build mobile apps using React Native.

Section 3: Back-End Development (4 months)

In the Back-End Development module of this course, participants will delve deeper into the intricacies of server-side programming, building upon the front-end skills acquired in the previous section. Throughout this module, students will gain comprehensive knowledge on designing, developing, and deploying server-side applications. They will become proficient in utilizing renowned back-end frameworks and technologies, including Node.js, Express, and either SQL or MongoDB, empowering them to create robust and scalable web applications. By the end of this module, students will have the skills necessary to bring their web development projects to life on the server side.

Topics covered in this section include:

  • Introduction to Back-End Development
  • Node.js and Express Framework
  • Database Design and Management
  • SQL or MongoDB
  • RESTful API Design
  • Authentication and Authorization
  • Deployment of Back-End Applications

Who is this course for?

  • Individuals who are interested in pursuing a career in web development and want to learn the necessary skills to become a full-stack web developer.
  • Beginners who have little to no experience in web development but have a strong desire to learn and are willing to put in the time and effort to develop their skills.
  • Anyone who is finding a chance to get the promotion

Requirements

  • No prior programming experience is necessary. This course is designed for beginners and will teach you everything you need to know.
  • A computer with internet access is required to complete the course.
  • No paid software is required. All of the tools and software used in the course are open source and freely available.
